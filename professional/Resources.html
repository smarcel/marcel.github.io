Software and Databases

Software

BOB (official successor of Torch3vision): a python and C++ library for signal processing, machine learning and biometrics


Torch3: a C++ machine learning library

Torch5: the new recent version of Torch

Torch3vision (no longer maintained): a C++ machine vision library based on Torch3

Annotations

Facial images

Annotations for Multi-PIE [HERE]: eye centers and multiple facial key points for face images from the Multi-PIE database


Annotations for MOBIO [HERE]: eye centers for face images from the MOBIO database (still images protocol)

Databases

Biometrics

MOBIO Mobile Biometrics database: audio-visual recordings of talking faces on a mobile device

Spoofing and Anti-Spoofing

PrintAttack: 200 video clips of printed-photo attack attempts to 50 clients, under different lighting conditions. It also contains 200 real-access attempt videos from the same clients


ReplayAttack: face spoofing consists of 1300 video clips of photo and video attack attempts to 50 clients, under different lighting conditions.


MaskAttack with Kinect (available soon) !

Gesture Recognition

Hand posture datasets for hand posture recognition (images)


Two-handed gesture dataset for hand gesture recognition (video sequences)


3D hand gesture dataset for hand gesture recognition (3D trajectories only)
