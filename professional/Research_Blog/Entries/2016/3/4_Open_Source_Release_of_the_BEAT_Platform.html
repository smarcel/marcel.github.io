BEAT Open Source Now

 
 

After 4 years of development, the BEAT platform is now available for download as an open-source project, under the Affero GNU GPL version 3. It is composed of 6 packages which can be reached on this gitlab group repository, hosted at Idiap:


https://gitlab.idiap.ch/beat


To get started, download the package "beat.web" and read its README.rst file located on its root for quick start instructions:


git clone https://gitlab.idiap.ch/beat/beat.web


In case of issues, please share your questions through our mailing list (https://groups.google.com/forum/#!forum/beat-devel).


The project totals ~80'000 lines of code split this way:

Python: 58%

Javascript: 24%

HTML: 15%

CSS: 3%


It makes heavy use of other open-source projects (such as Django or Twisted) and builds on the top of these frameworks to create the experience you'll find through the web application (e.g. https://www.beat-eu.org/platform/).


The Idiap instance of the platform is now updated with the software available on the link above. Let us know if you have issues accessing that link.
